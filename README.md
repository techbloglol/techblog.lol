# [Techblog.lol](https://techblog.lol)

A creative outlet for a group of tech-centric people.

To run a local instance:

    brew update
    brew install hugo
    git clone https://github.com/techbloglol/techblog.lol.git --recursive
    cd techblog.lol
    hugo server --watch
